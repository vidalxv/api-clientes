
## Documentação da API

#### Retorna todos os clientes

```http
  GET /clientes
```


#### Retorna um cliente

```http
  GET /clientes/${id}
```

| Parâmetro   | Tipo       | Descrição                                   |
| :---------- | :--------- | :------------------------------------------ |
| `id`      | `string` | **Obrigatório**. O ID do item que você quer |

#### Retorna todos as licencas

```http
  GET /licencas
```
#### Retorna todos os sistemas

```http
  GET /sistemas
```
